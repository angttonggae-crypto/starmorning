# 별모닝 🌟 (Byeol Morning)

매일 아침, 당신의 별자리 운세를 확인하고 기분 좋게 하루를 시작할 수 있도록 돕는 힐링 콘텐츠 웹서비스입니다. 별자리 입력 또는 원클릭 선택을 통해 오늘의 순위, 맞춤형 행동 요령, 그리고 랜덤 응원 메시지를 제공합니다.

---

## 1. 프로젝트 소개

- **서비스명**: 별모닝 🌟 (Byeol Morning)
- **목적**: 바쁜 현대인들에게 아침 시간 소소한 즐거움과 긍정적인 에너지를 전달하는 힐링 운세 서비스
- **기술 스택**: HTML5, CSS3, JavaScript (Vanilla JS)
- **배포 방식**: GitHub Pages를 통한 정적 웹사이트 배포 가능 형태

---

## 2. 주요 기능

- **별자리 선택 및 검색 기능**
  - 12개의 별자리 단추를 클릭해 즉시 조회 가능
  - 자동 완성/추천 기능이 포함된 별자리 검색 창 지원 (예: '양', '사수자리', '물고기' 등 유연한 동의어 처리)
- **사용자 입력 유효성 검증 (Validation)**
  - 공백 제출 시: `"별자리를 입력해주세요."` 경고창 노출 및 흔들림 애니메이션 효과
  - 올바르지 않은 단어 검색 시: `"올바른 별자리를 입력해주세요."` 경고창 노출 및 흔들림 애니메이션 효과
- **오늘의 별자리 순위 산출**
  - `"오늘의 ○○자리는 ○위입니다."` 포맷 제공
  - 날짜 기반 고유 시드(Date-seeded Hash) 알고리즘 적용: 하루 동안은 몇 번을 검색하더라도 동일한 별자리가 동일한 순위(1위~12위)로 매칭되어 운세의 신뢰성과 일관성 유지
- **순위 구간별 운세 메시지 및 추천/비추천 행동 요령 제공**
  - **1위 ~ 3위 (좋은 운세)**: 긍정적인 해석 + 자신감을 돋우는 실천 사항 제안
  - **4위 ~ 8위 (무난한 운세)**: 평온한 일상 유지 권장 + 소소한 행복 실천 제안
  - **9위 ~ 12위 (주의할 운세)**: 차분한 대처 요령 + 신중한 처신과 충분한 휴식 제안
- **조회 시마다 갱신되는 랜덤 응원 한마디**
  - 결과를 확인할 때마다 새로운 응원 문구가 무작위로 선택되어 사용자에게 기분 좋은 힐링 제공
- **다시 조회하기 기능**
  - 결과 화면에서 터치 한 번으로 부드러운 전환 효과를 통해 메인 검색 화면으로 복귀

---

## 3. 사용 방법

1. `index.html` 파일을 크롬, 사파리 등의 웹 브라우저에서 실행합니다.
2. 메인 페이지의 **12개 별자리 버튼** 중 자신의 별자리를 클릭하거나, **검색창**에 직접 별자리 이름(예: `사자자리` 또는 `사자`)을 입력하고 **[오늘의 운세 보기]** 버튼을 누릅니다.
3. 나타나는 결과 카드에서 오늘의 순위, 총평, 추천/비추천 행동 요령, 🌟오늘의 응원 한마디를 읽습니다.
4. 다른 별자리를 조회하거나 새로 검색하려면 카드 하단의 **[다시 조회하기 🔄]** 버튼을 누릅니다.

---

## 4. PRD 준수 구현 설명

이 프로젝트는 제공해주신 기획서(PRD)의 요구사항을 엄격하게 준수하여 설계 및 구현되었습니다.

- **HTML, CSS, JS만 사용**: 별도의 프레임워크나 복잡한 빌드 과정 없이 정적 파일(index.html, style.css, script.js)만으로 동작하여 GitHub Pages에 즉시 업로드하고 배포할 수 있습니다.
- **예외 처리 규칙 구현**: 사용자가 공백을 입력하거나 존재하지 않는 엉뚱한 값을 입력했을 때 각각 기획서에 명시된 에러 메시지(`"별자리를 입력해주세요."`, `"올바른 별자리를 입력해주세요."`)를 명확하게 띄워줍니다.
- **순위 구간 분기 처리**: 1~3위, 4~8위, 9~12위 구간을 철저히 구분하여 각각에 맞는 테마 색상(초록/파랑/빨강)의 배너와 분위기에 맞는 추천/비추천 행동 카드 리스트를 동적으로 렌더링합니다.
- **랜덤 응원 및 재시도**: 다시 조회하기 버튼이 유기적으로 메인 화면으로 돌려보내 주며, 동일한 별자리를 연속해서 조회하더라도 응원 한마디만큼은 실시간으로 랜덤 갱신됩니다.
- **디자인 컨셉 구현**: 아침 하늘의 차분하고 은은한 감성을 담은 다크-인디고/피치 그라데이션 배경, 서서히 깜빡이는 별빛 효과 애니메이션(twinkle), 그리고 반투명 글래스모피즘(Glassmorphism) 카드로 고급스러우면서도 귀여운 UI를 완성했습니다. 모바일 기기에서도 완벽히 1열 배치로 깨짐 없이 볼 수 있도록 반응형 웹 디자인(RWD)을 적용했습니다.

## 5. 향후 개선점 (Roadmap)

1. **날짜별 순위 변경 기능 고도화**
   - 현재 구현된 로컬 시간 연동 알고리즘을 확장하여 어제 운세 보기, 내일 운세 미리보기 등의 날짜 탐색 기능을 추가할 예정입니다.
2. **실제 운세 API 연동**
   - 현재의 일정한 규칙 기반 행운 산출 방식을 넘어서, 실제 외부 전문 운세 API나 인공지능 요약 API를 연동해 더욱 다채롭고 현실적인 개별 운세 텍스트를 제공하도록 개선할 예정입니다.
3. **사용자별 저장 기능 추가 (Favorites)**
   - 브라우저의 `localStorage` 또는 쿠키를 활용하여 사용자가 매번 검색하지 않아도 첫 진입 시 '나의 별자리'를 자동으로 로드하고 기록을 보관할 수 있는 개인화 기능을 추가할 계획입니다.
